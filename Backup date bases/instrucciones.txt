Lo que deberian hacer para cargar en sus DB es lo siguiente:
1- todo lo que es configurar postgresql (lo de la clase)

2- deben crear una base de datos vacia (preferecnialmente con el nombre "catalogo" asi evitariamos conflictos entre versiones); asi:
	createdb -U tu_usuario nombre_de_tu_base_de_datos

3- Y (segun chat gpt) deben restaurar el backup que les pase en esta carpeta y correr este comando:
	pg_restore -U tu_usuario -d nombre_de_tu_base_de_datos -v respaldo.bak
